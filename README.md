# Raspberry Pi Pico W

This repository is a practical workspace for Raspberry Pi Pico W development and experimentation.

It includes test projects, hardware connection references, code examples, and hands-on exercises focused on embedded systems learning.

The main goal is to document and organize:

- Hardware wiring and module connection experiments
- Basic and intermediate code examples
- Communication and integration tests

## Test Environment

- Development and testing are performed from a PC running a Linux distribution.

## Software Stack

- MicroPython is used to program the microcontroller.
- Development is done in Visual Studio Code.
- Raspberry Pi Pico extension for VS Code: [raspberry-pi.raspberry-pi-pico](https://marketplace.visualstudio.com/items?itemName=raspberry-pi.raspberry-pi-pico)
